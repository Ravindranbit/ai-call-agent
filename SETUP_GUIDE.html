1092 Assist: AI Voice Agent

v1.0.0 — Production Ready

This guide provides comprehensive instructions to deploy and run the 1092 Assist AI Voice Agent, a production-grade multilingual system built for the Karnataka Government Helpline.

Prerequisites

  • Node.js: Version 18.x or higher is required.
  • PostgreSQL: A running instance (Local or via Render/ElephantSQL).
  • Redis: Required for conversational session persistence.
  • Twilio Account: With an active phone number for Voice webhooks.
  • Groq & NVIDIA API Keys: For the hybrid Llama-3/Sarvam-M AI engine.

1. Environment Setup

Create a file named .env in the project root and configure the following parameters:

# Telephony Configuration
TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxxxxxxxxxxxxx
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_PHONE_NUMBER=+1234567890

# AI Models (Hybrid Engine)
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxxxxx
SARVAM_API_KEY=your_sarvam_api_key_here

# Database & Infrastructure
DATABASE_URL=postgresql://user:pass@host:5432/dbname
REDIS_URL=redis://default:pass@host:port
PUBLIC_URL=https://your-domain.com

2. Installation & Initialization

Run the following commands in sequence to install dependencies and prepare the environment:

# Install all required packages
npm install

# Build the project (if using TypeScript/Vite)
npm run build

3. Webhook Integration

To receive calls, you must point your Twilio phone number to the application endpoint:

  • Endpoint: {PUBLIC_URL}/api/call/incoming
  • Method: HTTP POST
  • Fallbacks: Ensure status callback is set to /api/call/status to track active calls accurately.

4. Execution Modes

The system supports both development and production execution modes:

Development

npm run dev

Enables hot-reloading and debug logging.

Production

npm start

Optimized performance with cluster support.

5. Operational Dashboard

Access the **Agent Panel** at /dashboard to monitor live telemetry:

  • Live Call Monitor: Real-time emotion and intent analysis.
  • Outbound Dialer: Trigger AI calls for live demonstrations.
  • Feedback Loop: Improve AI accuracy with human-in-the-loop corrections.
